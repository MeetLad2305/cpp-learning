# C++ Learning Repository

## About

Welcome to my C++ learning repository. This repository contains programs and mini-projects created while learning C++ programming concepts.

## Programs

### 1. Cab Fare Calculator

A program that calculates cab fare based on:

- Vehicle Type (Mini, Sedan, SUV)
- Distance Travelled
- Surge Pricing

### 2. Shipping Cost Calculator

A program demonstrating function overloading to calculate shipping costs using:

- Product Weight
- Distance
- Express Delivery Option

### 3. Product Billing System

A class-based application that:

- Stores product information
- Calculates total cost
- Applies discounts
- Displays product details in tabular format

## Concepts Covered

- Functions
- Function Overloading
- Default Arguments
- Classes and Objects
- Arrays of Objects
- Encapsulation
- Formatted Output using `iomanip`
