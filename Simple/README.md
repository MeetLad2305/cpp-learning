# Program 1: String Operations (Length, Palindrome, and Anagram Check)

## Description

This C++ program performs basic string operations using character arrays. It calculates the length of a string, checks whether the string is a palindrome, and determines whether two strings are anagrams.

## Concepts Used

* Character Arrays
* Loops
* Functions
* Frequency Array
* Palindrome Checking
* Anagram Checking
